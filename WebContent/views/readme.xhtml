JRuteros

Este README esta hecho con el unico objetivo de hacer saber, a aquella persona que pruebe la apicacion, que tanto el alta como la modificacion de ruta no redireccionan luego de completarse la operacion

Es decir que luego de efectuar el alta de una ruta para comprobar si la misma se realizo con exito puede ir a la pestaña de "Mis Rutas", donde deberia encontrarla

Despues de varios dias dedicados a encontrar la causa de este comportamiento se decidio por desistir, dado que no encontramos razon alguna para justificar el mismo. Algo que es todavia mas extraño es el hecho de que este comportamiento ocurre solo algunas veces

Desde ya pedimos disculpas por la desprolijidad
